## Run Locally

Clone the project

```bash
  git clone https://github.com/VasiliiOrekhov/NewsAnalayzer
```

Install dependencies

```bash
  npm
```

Start the dev varsion:

```bash
  npm run dev
```

Start the optimized (prod) version:

```bash
  npm build
```
